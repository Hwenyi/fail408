## 平面束方程
通过两条相交直线来定义平面，平面上的点既在这条线上，又在那条线上
设 ${\pi}_{i} : A_{i}x + B_{i}y + C_{i}z + D_{i} = 0,i = 1,2,A_{1},B_{1},C_{1}$ 与 $A_{2},B_{2},C_{2}$ 不成比例,则
过 $L : \left\{\begin{array}{l} A_{1}x + B_{1}y + C_{1}z + D_{1} = 0, \\ A_{2}x + B_{2}y + C_{2}z + D_{2} = 0 \end{array} \right.$ 的平面束方程为
$$
A_{1}x + B_{1}y + C_{1}z + D_{1} + \lambda\left( A_{2}x + B_{2}y + C_{2}z + D_{2} \right) = 0\text{ (不含 }{\pi}_{2}\text{ ),}
$$
或
$$
A_{2}x + B_{2}y + C_{2}z + D_{2} + \lambda\left( A_{1}x + B_{1}y + C_{1}z + D_{1} \right) = 0\text{(不含}{\pi}_{1}\text{).}
$$
