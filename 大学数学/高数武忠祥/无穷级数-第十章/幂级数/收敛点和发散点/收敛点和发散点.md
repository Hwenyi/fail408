# 收敛点
若给定 $x_0\in I$，有 $\sum_{n=1}^\infty u_n(x_0)$ 收敛，则称点 $x_0$ 为函数项级数 $\sum_{n=1}^\infty u_n(x)$ 的收敛点 

# 发散点
若给定 $x_0\in I$，有 $\sum_{n=1}^\infty u_n(x_0)$ 发散，则称点 $x_0$ 为函数项级数 $\sum_{n=1}^\infty u_n(x)$ 的发散点
