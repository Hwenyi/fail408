若 $\lim_{ x \to x_{0}^{-}  } f(x)=\infty$ 或 $\lim_{ x \to x_{0}^{+} }f(x) = \infty$，也就是说，该点，从左侧或者右侧逼近是无穷，无穷也是一种不存在，比如 $\frac{1}{x}$ 在 0 处;;; 无穷间断点
