## 题29
### 题目
【2012 统考真题】对下图所示的有向带权图,若采用 Dijkstra 算法求从源点 $a$ 到其他各项点的最短路径,则得到的第一条最短路径的目标顶点是 $b$ ,第二条最短路径的目标顶点是 $c$ ,后续得到的其余各最短路径的目标顶点依次是 ( ).
![](https://img.hwenyi.tech/202405291147178.webp)
A. $d,e,f$ 
B. $e,d,f$ 
C. $f,d,e$ 
D. $f,e,d$

### 解
C
从 $a$ 到各顶点的最短路径的求解过程下:
后续目标顶点依次为 $f,d,e$ 。
本题也可用排除法: 对于 $\mathrm{A}$ ,若下一个顶点为 $d$ ,路径 $a,b,d$ 的长度为 5,而 $a,b,c,f$ 的长度仅为 4,显然错误。同理可排除选项 B。将 $f$ 加入集合 $S$ 后,采用上述方法也可排除选项 D。

| 顶点  | 第 1 轮                   | 第 2 轮                     | 第 3 轮                       | 第 4 轮                       | 第 5 轮                       |
| --- | ----------------------- | ------------------------- | --------------------------- | --------------------------- | --------------------------- |
| $b$ | $( {a,b}) 2$ |                           |                             |                             |                             |
| $c$ | $( {a,c})$ 5 | $( {a,b,c}) 3$ |                             |                             |                             |
| $d$ | $\infty$                | $( {a,b,d}) 5$ | $( {a,b,d}) 5$   | $( {a,b,d}) 5$   |                             |
| $e$ | oo                      | $\infty$                  | $( {a,b,c,e})$ 7 | $( {a,b,c,e})$ 7 | $( {a,b,d,e}) 6$ |
| $f$ | oo                      | $\infty$                  | $( {a,b,c,f}) 4$ |                             |                             |
| 集合S | $\{ a,b\}$              | $\{ a,b,c\}$              | $\{ a,b,c,f\}$              | $\{ a,b,c,f,d\}$            | $\{ a,b,c,f,d,e\}$          |
|     |                         |                           |                             |                             |                             |
