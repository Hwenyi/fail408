## 题38
### 题目
已知系统为 32 位实地址,采用 48 位虚拟地址,页面大小为 $4\mathrm{{KB}}$ ,页表项大小为 $8\mathrm{\;B}$ 。假设系统使用纯页式存储, 则要采用 ( ) 级页表, 页内偏移 ( ) 位。
A. $3,{12}$ 
B. 3,14 
C. 4,12 
D. $4,{14}$
### 分析
页面大小为 $4\mathrm{{KB}}$ ,因此页内偏移为 12 位。系统采用 48 位虚拟地址,因此虚页号 ${48} - {12} = {36}$ 位。采用多级页表时,最高级页表项不能超出一页大小; 每页能容纳的页表项数为 $4\mathrm{{KB}}/8\mathrm{\;B} = {512} =$ ${2}^{9},{36}/9 = 4$ ,因此应采用 4 级页表,最高级页表项正好占据一页空间,所以本题选择选项 C。

### 解
C
