## 题36
### 题目
【2022 统考真题】下列关于 CPU 模式的叙述中, 正确的是 ( )。
A. CPU 处于用户态时只能执行特权指令
B. CPU 处于内核态时只能执行特权指令
C. CPU 处于用户态时只能执行非特权指令
D. CPU 处于内核态时只能执行非特权指令
### 分析
CPU 在用户态时只能执行非特权指令, 在内核态时可以执行特权指令和非特权指令。
### 解
C
