## 题28
### 题目
Q:【2015 统考真题】假定下列指令已装入指令寄存器, 则执行时不可能导致 CPU 从用户态变为内核态（系统态）的是 ( ) 。
```assembly
A. DIV R0, R1; (R0)/(R1) → R0
B. INT n; 产生软中断
C. NOT R0; 寄存器 R0 的内容取非
D. MOV R0, addr; 把地址 addr 处的内存数据放入寄存器 R0
```
### 分析
A:A只是进行一个数值计算，应该用不上
B产生中断了，可能是
C直接操作寄存器，这个汇编语言在用户态应该也可以做到
D写地址，高级语言在用户态都可以做到的，更何况汇编
我靠，选的是，不可能导致
那么对于A，可能产生除以0的异常，异常是要进入内核态的
D的话，感觉可能触发地址保护，也就是那个地方不能写数据
### 解
C
部分指令可能出现异常,从而转到核心态。指令 $\mathrm{A}$ 有除零异常的可能。
指令 $\mathrm{B}$ 为软中断[[指令]],用于触发一个[[中断]]并跳转到相应的[[中断处理程序]]," $\mathrm{n}$ " 表示pp,使用软中断可以在[[用户态]]和[[内核]]态之间切换,以实现[[系统调用]]。
指令 $\mathrm{D}$ 有缺页异常的可能。指令 $\mathrm{C}$ 不会发生异常。
