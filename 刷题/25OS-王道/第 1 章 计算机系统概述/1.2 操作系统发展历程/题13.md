## 题13
### 题目
操作系统有多种类型。允许多个用户以交互的方式使用计算机的操作系统，称为 ( ) ； 
允许多个用户将若干作业提交给计算机系统集中处理的操作系统，称为 ( ) ; 
在( )的控制下, 计算机系统能及时处理由过程控制反馈的数据, 并及时做出响应; 
在 IBM-PC 中, 操作系统称为 ( ) 。
A. 批处理系统 B. 分时操作系统
C. 实时操作系统 D. 微型计算机操作系统
### 分析
多用户交互，也就是分时
多作业，多任务就是批处理
及时处理也就是实时
IBM应该就进入了商业时代了，肯定也就是D了
### 解
B、A、C、D
这是操作系统发展过程中的几种主要类型。
