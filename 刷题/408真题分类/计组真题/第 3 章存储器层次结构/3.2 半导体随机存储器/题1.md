## 题1
### 题目
【2010】下列有关 RAM 和 ROM 的叙述中，正确的是（ ）。
Ⅰ. RAM 是易失性存储器，ROM 是非易失性存储器
Ⅱ. RAM 和 ROM 都采用随机存取方式进行信息访问
Ⅲ. RAM 和 ROM 都可用作 Cache
Ⅳ. RAM 和 ROM 都需要进行刷新
A. 仅 Ⅰ 和 Ⅱ
B. 仅 Ⅱ 和 Ⅲ
C. 仅 Ⅰ、Ⅱ 和 Ⅳ
D. 仅 Ⅱ、Ⅲ 和 Ⅳ
### 分析
现代存储器都是[[随机存取]]的，所谓的随机是指随意，我们常见的顺序读，只有读取了前面才能读后面的设备，是磁带
