## 题1
### 题目
【2017】某 C 语言程序段如下：
```cpp
for (i=0; i<=9; i++) {
    temp=1;
    for (j=0; j<=i; j++) temp*=a[j];
    sum+=temp;
}
```
下列关于数组 a 的访问局部性的描述中，正确的是（ ）。
A. 时间局部性和空间局部性皆有
B. 无时间局部性，有空间局部性
C. 有时间局部性，无空间局部性
D. 时间局部性和空间局部性皆无
### 分析
空间[[局部性]]是指相邻单位更快读取，时间局部性是指，常用的读得更快
