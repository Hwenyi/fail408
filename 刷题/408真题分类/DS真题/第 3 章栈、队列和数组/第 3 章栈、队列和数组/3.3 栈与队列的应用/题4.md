```cpp
int S(int n) { 
    return (n <= 0) ? 0 : S(n - 1) + n;
}
void main() { 
    cout << S(1);
}
```

A. main()→S(1)→S(0)
B. S(0)→S(1)→main()
C. main()→S(0)→S(1)
D. S(1)→S(0)→main()
