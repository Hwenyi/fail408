## 题22
### 题目
> [!question]+
> 下列选项中，能引起外部中断的事件是（ ）。
> 
> A. 键盘输入
> 
> B. 除数为 0
> 
> C. 浮点运算下溢
> 
> D. 访存缺页
### 解
> [!done]+
> 外部中断：指 CPU 执行指令以外的事件产生的中断。一般是指由计算机外设发出的中断请求，如：键盘中断、打印机中断、定时器中断等。根据中断源是否可以被屏蔽，外中断可分为两类：
> 
> - 可屏蔽外中断：可屏蔽外中断是 CPU 可以不响应的外中断。CPU 是否响应可屏蔽中断，要看标志寄存器 IF 位的设置。当 CPU 检测到可屏蔽中断信息时，如果 IF=1，则 CPU 在执行完当前指令后响应中断，引发中断过程；如果 IF=0，则不响应可屏蔽中断。几乎所有由外设引发的外中断，比如键盘输入、打印机中断等，都是可屏蔽中断。
> - 不可屏蔽外中断：不可屏蔽外中断是 CPU 必须响应的外中断。当 CPU 检测到不可屏蔽中断信息时，则在执行完相亲指令后，立即响应，引发中断过程。比如突然掉电等。
> 
> 内部中断：指 CPU 执行指令以内的事件产生的中断。主要指程序性异常，例如运算除数为零、单步中断等、执行 int0 指令、执行 int0 指令等所引起的中断。
> 
> A 正确。当用户从键盘输入数据时，计算机系统可以通过中断请求（IRQ）机制接收到键盘控制器发送的中断信号，从而响应并处理键盘输入。因此，键盘输入可以引起外部中断。
> 
> B 错误。除数为 0 属于内部中断，为运行时错误，但通常不会引发外部中断，而是由处理器内部的异常机制（如浮点异常或算术异常）处理并触发异常中断。
> 
> C 错误。浮点运算下溢也属于内部中断，为运行时错误，可以由处理器内部的浮点异常机制处理。
> 
> D 错误。访存缺页属于内部中断，为一个运行时错误，但它通常由操作系统的内存管理单元（MMU）或页面错误中断（Page Fault Interrupt）来处理。
> 
> 综上，A 为外部中断，B、C、D 为内部中断。
> 
> 本题选 A。
> 
> 评：内中断和外中断有很多不同的划分标准，有时候内中断也表示内部异常，外中断也表示外部异常，具体根据上下文判断。一般情况下，内中断指 CPU 执行指令以内的事件产生的中断，外中断主要指 I/O 中断。
