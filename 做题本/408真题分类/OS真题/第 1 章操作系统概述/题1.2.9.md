## 题1.2.9
### 题目
【2015 统考真题】假定下列指令已装入指令寄存器, 则执行时不可能导致 CPU 从用户态变为内核态（系统态）的是 ( ) 。
```assembly
A. DIV R0, R1; (R0)/(R1) → R0
B. INT n; 产生软中断
C. NOT R0; 寄存器 R0 的内容取非
D. MOV R0, addr; 把地址 addr 处的内存数据放入寄存器 R0
```
### 分析
除以0属于是[[故障]]，故障是[[异常]]的一种，异常就是要进入[[内核]]的，D既然在访存，我觉的也是要得，B的软中断是什么意思？我最后选C
### 解
C
对于A选项中的指令，可能会出现零作为除数的情况而导致中断。
B选项中的指令为中断指令，而中断处理 必须在内核态下执行。
对于C选项中的指令，取非操作不会产生异常。
对于D选项中的指令，可能会产生缺页中断。所以正确答案为 C 选项 。
