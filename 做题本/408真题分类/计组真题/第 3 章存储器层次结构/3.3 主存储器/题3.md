## 题3
### 题目
【2011】某计算机存储器按字节编址，主存地址空间大小为 64MB，现用 4M×8 位的 RAM 芯片组成 32MB 的主存储器，则存储器地址寄存器 MAR 的位数至少是（ ）。
A. 22 位
B. 23 位
C. 25 位
D. 26 位
### 分析
[[MAR]]是指主存空间的寻址范围，[[MDR]]是指每次寻的小单位，这里说的[[内存|RAM]]的大小其实干扰选项，它对MAR没有影响，因为哪怕单个ram不够大，我们也可以通过拓展，让大达到我们想要的主存大小
