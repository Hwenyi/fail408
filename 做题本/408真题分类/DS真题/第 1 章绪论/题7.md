## 题7
### 题目
【2022】下列程序段的时间复杂度是（ ）。
```cpp
count = 0;
for (i = 1; i < n; i *= 2) 
    for (j = 0; j < i; j++) 
        count++;
```
A. $O(\log_2n)$
B. $O(n)$
C. $O(n\log_2n)$
D. $O(n^2)$
### 分析
