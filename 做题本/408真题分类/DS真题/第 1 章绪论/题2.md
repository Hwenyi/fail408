### 题目
【2012】求整数 $n(n\ge0)$ 阶乘的算法如下，其时间复杂度是（ ）。
```cpp
int fact(int n) {
    if (n <= 1) return 1;
    return n * fact(n - 1);
}
```
A. $O(\log n)$
B. $O(n)$
C. $O(n\log n)$
D. $O(n^2)$
### 分析
递归的[[时间复杂度]]，先看终止条件，再看每次进入递归的时候，我们如何操作终止条件的变量，看看它到终止条件需要运行多少次
