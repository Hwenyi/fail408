## 题31
### 题目
Q:对信号量 $S$ 执行 $\mathrm{P}$ 操作后,使该进程进入资源等待队列的条件是 ( ) 。
A. S. value $< 0$ 
B. S. value $< = 0$
C. S. value $> 0$ 
D. S. value $> = 0$
### 分析
A:P操作是指信号量的值减一，如果信号量的值小于0，表示资源不可用，有进程在等待资源。 
要让P操作可以有东西减，所以得是正数
### 解
A
参见**记录型信号量**的解析。此处极易出 S. value 的物理概念题, 现在总结如下:
① S.value $> 0$ ,表示某类可用资源的数量。每次 $P$ 操作,意味着**请求分配一个单位的资源**。
② S.value $\leq 0$ ,表示某类资源已经没有,或者说**还有因请求该资源而被阻塞**的进程, S.value 的绝对值表示等待进程数目。
一定要看清题目中的陈述是执行 $P$ 操作前还是执行 $P$ 操作后。
