## 题20
### 题目
Q:一个作业 8:00 到达系统，估计运行时间为 $1\mathrm{\;h}$ 。
若 10:00 开始执行该作业，其响应比是 ( ) 。
A. 2 
B. 1 
C. 3 
D. 0.5
### 分析
A:响应比 = (等待时间 + 要求服务时间) / 要求服务时间
这里的等待时间是指，到达系统的时间 到 开始执行任何的时间
这里的 要求服务时间是指估计运行时间
### 解
C
$$
\text{响应比} = \frac{\text{ 等待时间 } + \text{ 要求服务时间 }}{\text{ 要求服务时间 }} = \frac{2 + 1}{1} = 3
$$
