## 题49
### 题目
进程创建时，不需要做的是 ( ) 。
A. 填写一个该进程的进程表项 B. 分配该进程适当的内存
C. 将该进程插入就绪队列 
D. 为该进程分配 CPU
### 分析
进程创建原语完成的工作是: 向系统申请一个空闲 PCB, 为被创建进程分配必要的资源, 然后将其 PCB 初始化, 并将此 PCB 插入就绪队列, 最后返回一个进程标志号。
当调度程序为进程分配 CPU 后, 进程开始运行。所以进程创建的过程中不会包含分配 CPU 的过程, 这不是进程创建者的工作, 而是调度程序的工作。
### 解
D
