## 题41
### 题目
在以下描述中, ( ) 并不是多线程系统的特长。
A. 利用线程并行地执行矩阵乘法运算
B. Web 服务器利用线程响应 HTTP 请求
C. 键盘驱动程序为每个正在运行的应用配备一个线程, 用以响应该应用的键盘输入
D. 基于 GUI 的调试程序用不同的线程分别处理用户输入、计算和跟踪等操作
### 分析
整个系统只有一个键盘, 而且键盘输入是人的操作, 速度比较慢, 完全可以使用一个线程来处理整个系统的键盘输入。
### 解
C
