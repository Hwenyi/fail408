## 题28
### 题目
Q:采用分页或分段管理后, 提供给用户的物理地址空间 ( ) 。
A. 分页支持更大的物理地址空间
B. 分段支持更大的物理地址空间
C. 不能确定
D. 一样大
### 分析
A:物理空间不变，这是一种虚拟化的手段
哦哦哦，对，物理结构上的绝对大小肯定是不变的，但是索引本身这个表还是在内存里占着空间的，所以是C
### 解
C
页表和段表同样存储在内存中, 系统提供给用户的物理地址空间为总空间大小减去页表或段表的长度。
由于页表和段表的长度不能确定, 所以提供给用户的物理地址空间大小也不能确定。
