## 题16
### 题目
Q:在 I/O 接口中设置中断触发器保存外设发出的中断请求, 是因为 ( )。
A. 中断不需要立即处理
B. 中断设备的处理速度比 CPU 快
C. CPU 无法对发生的中断请求立即进行处理
D. 可能有多个中断同时发生
### 分析
A:思考的角度是，每一个中断请求都是带指令周期的末尾去响应的。这就是这个无法立即处理的本意。
### 解
C
因为 CPU 无法对发生的中断请求立即进行处理, 因此需要在 I/O 接口中设置中断触发器, 以保存是哪些外设发出了中断请求, 等 CPU 当前的指令周期结束后, 响应中断并进行处理。
