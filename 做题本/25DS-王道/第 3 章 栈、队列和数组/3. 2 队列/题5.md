## 题5
### 题目
Q:循环队列存储在教组 `A[0...n]` 中,入队时的操作为 ( ).
A. `rear = rear + 1` 
B. `rear = (rear + 1) mod (n - 1)`
C. `rear = (rear + 1) mod n`
D. `rear = (rear + 1) mod (n + 1)`
### 分析
A:注意下标是0，一共有n+1个元素
### 解
D
数组下标范围 $0 \sim  \mathrm{n}$ ,因此数组容量为 $\mathrm{n} + 1$ 。循环队列中元素入队的操作是 `rear = (rear + 1) mod maxsize` , 题中 `maxsize=n+1` 。
因此入队操作应为 
```cpp
rear = (rear + 1) mod (n + 1)
```
