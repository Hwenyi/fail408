## 题32
### 题目
![](https://img.hwenyi.live/202408250121305.webp)
【2016 统考真题】已知一个带有表头结点的双向循环链表 $L$ ,结点结构为 prev $\mid$ data $\mid$ next , 其中 prev 和 next 分别是指向其直接前驱和直接后继结点的指针. 现要删除指针 $\mathrm{p}$ 所指的结点,正确的语句序列是 ( ).
A. 
```cpp
p->next->prev = p->prev;
p->prev->next = p->next;
free(p);
```
B. 
```cpp
p->next->prev = p->next;
p->prev->next = p->next;
free(p);
```
C. 
```cpp
p->next->prev = p->next;
p->prev->next = p->prev;
free(p);
```
D. 
```cpp
p->next->prev = p->prev;
p->prev->next = p->next;
free(p);
```
### 分析
![](https://img.hwenyi.live/202408250146293.webp)
### 解
D
$\mathrm{A}$ 的第二句代码,相当于将 $\mathrm{p}$ 前驱结点的后继指针指向其自身,错误; 
$\mathrm{B}$ 和 $\mathrm{C}$ 的第一句代码, 相当于将 $\mathrm{p}$ 后继结点的前驱指针指向其自身,错误。只有 $\mathrm{D}$ 正确。
