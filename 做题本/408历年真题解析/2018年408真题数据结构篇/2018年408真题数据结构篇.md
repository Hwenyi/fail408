# 后记

2018 年 408 真题总体难度偏低，一般所以 408 呈现大小年交替规律，也是奇数年相对简单，偶数年相对难，但没有数学那样明显。虽然 2023 大概率是小年，但是 408 的总体难度是逐年上升的，此小年非彼小年。

2018 年一位本科专业为食品工程的学长跨考浙大计算机，408 单科考出了 148 分这种前无古人目前后无来者的分数，强势上岸。说实话，跨考是一件非常不容易的事情，比如我这种零基础还妄想一口气吃个胖子的就直接被虐的渣都不剩，作为对比，学长是把四门专业课的教材翻烂了、配套的课后习题一题不漏地做完并吃透了再上考场的，这种学习专业课的细致程度完全超越了很多计算机科班学生。

如果能把事情做到极致，跨考什么的似乎也不是什么难事。

**下定决心，不怕牺牲，排除万难，去争取胜利。**

最后，祝大家国庆节快乐！
